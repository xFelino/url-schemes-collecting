URL schemes collecting
---

收集 iOS 常用 App 的 URL schemes，可用于各种启动器，或者添加到桌面。

欢迎`Star`、`Pull request`或提交 `Issue`。

#### 1. 微信
功能 | URL | 备注
---- | ---- | ----
扫一扫 | weixin://scanqrcode
朋友圈 | weixin://dl/moments
收藏 | weixin://dl/favorites
我的二维码 | weixin://dl/myQRcode
